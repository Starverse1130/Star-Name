# ⭐ Star-Name

A beautiful and interactive web application that creates a personalized "Star name" experience. This project demonstrates HTML, CSS, and JavaScript fundamentals with a clean, user-friendly interface.

## ✨ Features

- **Interactive User Interface**: Clean and modern design with a centered card layout
- **Name Input**: User-friendly text input field with placeholder text "Join Starverse"
- **Responsive Design**: Optimized for different screen sizes with flexbox layout
- **Hover Effects**: Smooth button hover animations for better user experience
- **Image Display**: Profile image display with styled borders and rounded corners
- **Professional Styling**: Custom CSS with gradient backgrounds and shadow effects

## 🚀 How to Use

1. **Clone or Download** the project to your local machine
2. **Open** `index.html` in your web browser
3. **Enter** your name in the input field
4. **Click** the "Star" button to generate your star name
5. **View** the personalized result

### To Run Locally

Simply double-click `index.html` or open it with your preferred web browser.
